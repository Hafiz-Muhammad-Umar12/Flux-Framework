## Features

- Provider-agnostic (Ollama, OpenAI, Anthropic)
- Async-first with sync wrappers
- Protocol-based (no inheritance required)
- Composable middleware
- Event-driven observability
- Tool system with @tool decorator
- Agent-to-agent handoffs
- Input/output guardrails
- Session persistence (InMemory, SQLite)
- Long-term memory
- Streaming responses
- Console and file tracing
